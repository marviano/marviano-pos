# Slideshow Images

This folder contains promotional images for the customer display slideshow.

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 1920x1080 (Full HD) or 1366x768 (HD)
- **Aspect Ratio**: 16:9 (landscape)
- **File Size**: Under 2MB for optimal performance

## Naming Convention

Use descriptive names for easy management:

```
promotion-001-summer-special.jpg
promotion-002-ice-cream-delight.png
promotion-003-lemon-drinks.webp
promotion-004-momoyo-brand.jpg
```

## Automatic Detection

The slideshow system will automatically detect and load all images from this folder. No configuration needed!

## Adding New Images

1. Add your image files to this folder
2. Use descriptive names following the convention above
3. Restart the app to see new images in the slideshow
4. Images will rotate automatically every 5 seconds

## Supported Formats

- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif` (static images only)
